Fitness Exercise Gallery

Site Name

Fitness Exercise Gallery

This name clearly communicates the site's purpose: a gallery of exercises categorized by muscle group. It's easy to remember, descriptive, and suitable for fitness enthusiasts looking for exercise ideas and instructions.

Optional domain availability: fit-exercise-gallery.com

Site Purpose

The Fitness Exercise Gallery provides a visual and interactive guide to strength training exercises, organized by muscle group. Each gallery features multiple exercises with step-by-step photos, embedded demonstration videos, and clear instructions, helping users learn proper form and discover new workouts.

Scenarios

  • Which exercises can I do to target my chest muscles, and how do I perform them correctly?
  • Can I see a demonstration video and step-by-step images for each exercise?

Color Schema

  • Primary Blue (#1a73e8): Used for headings, buttons, and accents.
  • Accent Yellow (#fbbc04): Used for highlights, icons, and call-to-action elements.
  • Background Gray (#f5f7fa): Used for page background.
  • Text (#222): Used for main body text.

Typography

  • Roboto (sans-serif): Used for all headings and body text for a clean, modern look.
  • Arial (fallback): Used if Roboto is unavailable.

Wireframe

Each exercise contains two images (toggle or swipe), an embedded YouTube video, and a description.
